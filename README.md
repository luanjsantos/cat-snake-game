# 🐱 Cat Snake Game

A snake game with a cat theme built with **TypeScript + Vite**.  
Control a cat 🐱 collecting fish 🐟 — don't hit the walls or yourself!

## Getting Started

**Node.js 22 LTS** recommended. Minimum: 18.

```bash
node --version  # v22.x.x
npm install
npm run dev
```

## Controls

| Key | Action |
|---|---|
| `↑` `↓` `←` `→` | Move |
| `W` `A` `S` `D` | Move |
| `Enter` / `Space` | Start / Restart |

## Features

- 🐱 Cat head + colorful body
- 🐟 Fish as food
- Score & best score tracking
- Smooth dark UI
- No dependencies (pure Canvas API)

## Tech Stack

- TypeScript
- Vite
- HTML5 Canvas
